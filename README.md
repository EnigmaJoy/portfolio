# Portfolio — Joselyn Grace Gordillo Lopez

Personal portfolio website built with Vue 3, Vuetify 3, and vue-i18n.
Live at: **[enigmajoy.github.io/portfolio](https://enigmajoy.github.io/portfolio)** *(coming soon)*

---

## Stack

- **Vue 3** — Composition API + `<script setup>`
- **Vuetify 3** — UI components
- **vue-i18n** — Multilingual support (IT · EN · ES · ZH)
- **Vue CLI** — Build tooling

## Features

- Animated cursor trail with section-aware color palettes
- Auto-scrolling photo carousel with drag/swipe support
- Skills grid with tech icons
- Project cards linked to GitHub
- Contact form (mailto)
- Language switcher with localStorage persistence

## Sections

| Section | Color | Content |
|---|---|---|
| Home | gold | Name + navigation |
| Welcome | purple | Skills & technologies |
| About me | blue | Bio + photo carousel |
| Projects | pink | GitHub project cards |
| Contact | gold | Form + social links |

## Run locally

```bash
npm install
npm run serve      # dev server → localhost:8080
npm run build      # production build → /dist
```
